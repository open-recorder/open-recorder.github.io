An ultra-lightweight, legacy-styled screen recorder built natively for Debian, Ubuntu, and Linux Mint.

Download .deb Package

Built for Gaming

Supports fluid 60 FPS recording and high-bitrate "Ultra" quality configurations. Capture fast-paced gameplay without missing detail.

Zero Bloat

Written cleanly in native Python 3 and PyQt6. No background tracking, heavy web frameworks, or annoying popups.

Audio Capture

Includes single-click toggles for system and desktop audio using native PulseAudio backends.

Quick Installation

Download the official OpenRec.deb bundle, navigate to your download location, and run the system package manager:

sudo apt update
sudo apt install ./OpenRec.deb
